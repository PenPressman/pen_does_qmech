About this project

These applets follow the structure of Griffiths' Introduction to Quantum Mechanics (2nd/3rd ed.), the standard undergraduate text. Each applet is self-contained HTML/CSS/JS — no build tools, no dependencies, no framework. Just open in a browser.

Physics is computed analytically where possible (exact eigenstates, ladder operators, spherical harmonics via recurrence), and numerically via finite-difference TDSE evolution for the wave packet applet.

Built with
  • Vanilla HTML · CSS · JavaScript
  • Canvas 2D API for rendering
  • Zero build tools or bundlers
  • Zero runtime dependencies
